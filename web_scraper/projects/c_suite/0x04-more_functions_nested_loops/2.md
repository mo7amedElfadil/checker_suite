# 2. Collaboration is multiplication



Write a function that multiplies two integers.

Prototype: int mul(int a, int b);

```julien@ubuntu:~/0x04$ cat 2-main.c
#include "main.h"
#include <stdio.h>

/**
 * main - check the code

 * Return: Always 0.

{
    printf("%d\n", mul(98, 1024));
    printf("%d\n", mul(-402, 4096));
    return (0);
}
```
